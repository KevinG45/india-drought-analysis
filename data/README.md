# Data Directory

Due to GitHub's file size limitations, the complete datasets are hosted on Kaggle:

🔗 **Full Dataset**: https://www.kaggle.com/datasets/kmgs04/india-drought-analysis-data-2000-2023

## Quick Download

### Using Kaggle API:
```bash
pip install kaggle
kaggle datasets download -d kmgs04/india-drought-analysis-data-2000-2023
```

### Manual Download:
Visit the link above and click "Download" button.

## Data Structure

Once downloaded, place files in this directory:
- data/raw/ - Original downloaded files
- data/processed/ - Cleaned and processed files

## Sample Data

Small sample files for testing are available in the Kaggle dataset.
